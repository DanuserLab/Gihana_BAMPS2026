# Gihana_BAMPS2026

This repository contains the key scripts and parameters used for protrusion enrichment analysis in Gihana et al., 2026 (Figure 2).

`ggRasAnalysis.m` is the primary analysis script.

- `mainFolder`: Directory containing all cells of interest. Each cell should include three channels corresponding to the GEF domain, KRAS domain, and membrane marker.
- `savePath`: Directory where output plots will be saved.
- `mySegParameters`: Path to the segmentation parameter file.

`tiamLabels.csv` contains the relevant segmentation parameters used in the analysis.
